# tui-rpg
A simple TUI RPG.

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (1.70+)

## Setup

```bash
# Install Rust (if not installed)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Initialize project
cargo init

# Build
cargo build
```

## Run

```bash
cargo run
```
