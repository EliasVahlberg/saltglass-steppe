# 7 Content Production Kits (Templates for Designers)

---

Audience: Level designers, content designers, narrative implementers Owner: Content Lead Purpose: Speed up production with repeatable patterns.

Split into small, highly-usable docs:

- Site Kit: dungeon archetype template (entrances, loops, hazards, reward beats)
- Encounter Kit: encounter tables with "twists" (storm-triggered variants)
- Rumor/Contract Kit: how a rumor becomes a quest chain
- Relic Kit: item categories, affix rules, lore snippet formats

Outputs: Plug-and-play content that still feels authored.
