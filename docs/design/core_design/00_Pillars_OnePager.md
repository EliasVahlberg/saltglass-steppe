# 0 Project Pillars (One-Pager)

---

Audience: Studio leadership, creative director, production, everyone Owner: Creative Director + Lead Producer Purpose: Align the studio in 5 minutes; prevent scope drift.

Contents:

- High concept + player fantasy (2--3 sentences)
- 3--5 pillars (e.g., "Storms rewrite maps," "Mutation with social consequences," "Readable light tactics")
- Target platforms, target session length, roguelike structure assumptions
- What the game is *not* (non-goals)
- Vertical slice definition (what "done" means)

Outputs: Approved pillars + success criteria for the slice.
